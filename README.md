# Finance Tracker Application

A full-stack Finance Tracker web application built using Flask and MySQL.
Users can track income and expenses, view charts, and manage their financial history securely.

## Features
- User Registration & Login
- Add, Edit, Delete Transactions
- Income & Expense Chart Visualization
- Monthly Summary Dashboard
- Secure Authentication using Sessions
- MySQL Database Integration

## Tech Stack
- Backend: Flask (Python)
- Frontend: HTML, CSS, Bootstrap, JavaScript, Chart.js
- Database: MySQL

## How to Run Locally

1. Create Virtual Environment
   python -m venv venv
   venv\Scripts\activate

2. Install Dependencies
   pip install -r requirements.txt

3. Run Application
   python app.py

4. Open in Browser
   http://127.0.0.1:5000
